我们再来看一下使用 static 和 register 修饰的输出结果

这回输出的为 a:20、 b:60 这个又是为什么呢？我们看下生成的 C++代码：

通过 C++ 代码我们发现，变量 b 使用 static 修饰结构体 main_block_impl_0 保存b 变量的指针，构造函数传递的该变量的指针，所以输出的值会发生改变。而通过 register 修饰的变量 a，结构体 main_block_impl_0 保存 a 的值，所以值会发生改变。

总结：

变量类型 捕获到block内部 访问方式
auto（局部变量） ✅ 值传递
static（局部变量） ✅ 指针传递
全局变量 ❌ 直接访问
